# NLP — French generation

Two modes, both producing **grammatical French** describing each damage and an overall report.

## Mode 1 — [Template engine](templates.md)

The **production path**. Deterministic French generator.

- Takes the structured outputs of Stages 0/1/2 + fusion.
- Uses 4 hand-crafted French lexicons (parts, damages, severity, position).
- Outputs sentences with correct gender agreement (e.g. *"casse importante"* vs *"enfoncement important"*).
- Audited by unit tests.

## Mode 2 — [CNN+LSTM captioner](captioner.md)

The **end-to-end** path. A small image-captioning network.

- MobileNetV3-Small encoder + LSTM decoder (Show-and-Tell style).
- Trained via **knowledge distillation** from the template engine.
- Generates French directly from the image — no pipeline needed at inference.
- Demonstrates end-to-end representation learning.

## Why both

| Question | Mode 1 | Mode 2 |
|---|---|---|
| Always grammatically correct? | yes | mostly |
| Needs the pipeline at inference? | yes | no |
| Auditable? | yes | no |
| What you'd ship in production | yes | maybe later |
| What proves you understand deep learning | no | yes |

Both modes share the **same training signal**: Mode 2's targets are sentences generated by Mode 1 from real pipeline outputs. So Mode 1 is also the teacher in the distillation setup.
